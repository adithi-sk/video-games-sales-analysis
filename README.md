🎮 **Video Game Sales Analysis**

📌 **_Project Overview_**

This project analyzes global video game sales data to identify trends across platforms, genres, and regions using:

- Excel
- Python (Google Colab)
- Power BI

---

📊 **_Key Insights_**

- Identified top-selling video games globally
- Analyzed sales based on genre. Action and Sports genres dominate global sales
- Compared sales across regions (NA, EU, JP, Others). NA generates the highest video game sales
- Analyzed platform-wise performance. PS2 and Xbox360 have the highest sales
- Observed trends over time. Sales peaked during early 2000s–2010

---

🛠 **_Tools Used_**

- Microsoft Excel (Data Cleaning, Pivot Tables)
- Python (Pandas, Matplotlib - Google Colab)
- Power BI (Interactive Dashboard)

---

📁 **_Project Structure_**

* `data/` → Raw dataset
* `excel/` → Excel analysis
* `colab/` → Python notebook
* `powerbi/` → Dashboard file
* `images/` → Visual outputs
