🎮 **Video Game Sales Analysis**

📌 **_Project Overview_**

This project analyzes global video game sales data using:

- Excel
- Python (Google Colab)
- Power BI

---

📊 **_Key Insights_**

- Identified top-selling video games globally
- Compared sales across regions (NA, EU, JP, Others)
- Analyzed platform-wise performance
- Observed trends over time

---

🛠 **_Tools Used_**

- Microsoft Excel (Pivot Tables, Dashboard)
- Python (Pandas, Matplotlib)
- Power BI (Interactive Dashboard)

---

📁 **_Project Structure_**

* `data/` → Raw dataset
* `excel/` → Excel analysis
* `colab/` → Python notebook
* `powerbi/` → Dashboard file
* `images/` → Visual outputs
